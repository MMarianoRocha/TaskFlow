# TaskFlow

## Visão Geral

O Task App Group é uma aplicação web local para gerenciamento

de tarefas colaborativas, executada em ambiente local (localhost),

sem exposição externa de rede, adequada para ambientes corporativos

com restrições de segurança.

Cada usuário possui autenticação própria e pode:

- Criar tarefas
- Visualizar suas atividades
- Compartilhar tarefas com outros usuários
